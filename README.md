
# War of the Ring Bot

Web application created for solo play created from Queller Bot flow charts.

https://wotr.infinityfreeapp.com/

Warriors of Middle Earth expansion now available!!

Being a web app, you can use browser translation feature to translate to almost any language!

## Versions

### Modern Edition (NEW!)
A modern static web application built with React, TypeScript, and Tailwind CSS.
- Located in `app/` directory
- Non-scrolling, screen-based UI
- Modern development workflow with Vite
- See [app/README.md](app/README.md) for details

### Legacy Edition
The original version using plain HTML/CSS/JavaScript.
- Located in `htdocs/` directory
- Scrolling transcript interface
- Works directly in browser without build step

## Development

### Modern App
```bash
cd app
npm install
npm run dev
```

### Legacy App
Simply open `htdocs/index.html` in a web browser.

## Story Content

Use Inky editor to drive the scripts choice branches, available at:

https://github.com/inkle/inky

Then export story.js to use with rest of web page.
